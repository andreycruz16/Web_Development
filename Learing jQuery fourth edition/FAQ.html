DATA COMMUNICATION AND NETWORKS




A Computer network is a number if computers interconnected by one or moretransmission paths. The transmission path often is the telephone line, due to itsconvenience and universal preserve



Data Communication is the exchange of data (in the form of Os and 1s) between twodevices via some form of transmission medium (such as a wire cable)



The purpose of data communication is to exchange information between two agents.



Data Communication is considered
Local - if the communicating device are in the same building.
Remote - if the device are farther apart.
